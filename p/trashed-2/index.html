把百度网盘文件转存到onedrive

Apr 21, 2020
阅读时长: 1 分钟

2020.5.2更新，今天试了了个新办法，在本地转存的基础上改了一下：购买一台vps，必须海外的（onedrive上传快，带宽高），然后一键安装win系统挂载onedrive网盘，再安装百度云直接下载到挂载盘上去，速度还不错，也不占用自己的电脑。vps商家我推荐vultr:https://www.vultr.com     ，因为你流量用完了还可以重置流量。推荐用本文链接注册，注册送100美金链接：https://www.vultr.com/?ref=8037199   等有时间出个教程吧。

百度网盘限速是众所周知的，以前还不承认。4.17又把Pandownload作者弄进去了，时隔三天，又爆出百度网盘用户激励计划  真的是百度网盘面前众生品等啊！一家独大带来的垄断地位，势必会压榨用户为自己获取更多利益。 水能载舟亦能覆舟，希望百度能意识到这一点，这里教大家如何把百度网盘的文件传输到onedrive网盘。

官方办法

用百度网盘开SVIP一个月，同时上传onedrive。

通过第三方服务商

multcloud

通过转存,这个方法试用很多网盘

官网：https://www.multcloud.com/

multcloud支持googledrive、onedrive、dropbox、百度网盘、webdav等主流云存储，甚至也支持owncloud这种自建网盘（见上图）。使用方法也很简单：添加相应的网盘，授予权限，选中文件转存到其他网盘。其原理应该是用自己的服务器作上传下载中转。每个月免费额度只有30G，速度也很慢。会员又很贵，有这钱还不如用自己服务器传。

Mover

免费并且不限制网盘数量、不限制流量的，可以自动跳过相同的目录以及重复文件。现在已被微软收购。

官网：https://mover.io/

支持将“Mover Agent / Amazon S3 / Amazon WorkDocs / Microsoft Azure / Box / Dropbox / Egnyte / FTP / Google Drive / Office 365 / OneDrive / SFTP / SharePoint Online / WebDAV”中的文件迁移到自己的服务：“Azure Blob Storage / Office 365 (OneDrive/SharePoint Admin) / OneDrive Consumer / OneDrive for Business (Single User)”中。

用软件通过自己电脑传

Air Explore

软件下载地址：点击下载

支持OneDrive、WebDAV和Google Drive等绝大多网盘

通过服务器传

安装及配置自动上传脚本

参考《Aria2 + Rclone 实现 OneDrive、Google Drive 等网盘离线下载》

设置第三方百度网盘工具

如果你对 RPC 设置还不了解，参考《使用 Aira2 下载文件后上传到 OneDrive 网盘》

以速盘（Speed­Pan）为例，来进行设置，其它工具的操作都是类似的，无非就是 填写 IP 地址、端口、密码，非常简单。直接看图：

注意事项： 下载路径必须设定成和 VPS 上的 Aria2 配置文件中的下载路径一致，才能正常触发自动上传脚本。

完成以上操作，通过速盘下载的文件就自动上传到onedrive了。

 

这里新手推荐Air Explore，不过要注意的是我们家里的网速属于不对等带宽，例如100M网络，上传速率只有20M左右。同步需要电脑开机很长时间。

有国外VPS的可以尝试用速盘，国内的VPS就算了，带宽太低，不知道要下载到猴年马月。
