Centos 7 安装 Aira2配合 KodExplorer 实现离线 BT 下载

JApril 03, 20030
阅读时长: 1 分钟

自从搭建了自己的服务器，总不忍让它吃灰，只放一个可道云太可惜了，于是想着搭建给可道云加一个支持bt的离线下载，网上找了很多教程，但是可能是自己主机的原因，直接卡死在安装Aria2这一步，不过还好这个教程可用。

##1、安装 Aria2 yum install aria2 ##2、生成配置文件并编辑 mkdir /etc/aria2/ vi /etc/aria2/aria2.conf #创建并编辑配置文件

直接粘贴后修改 rpc-secret=设置你的秘钥 #允许rpc enable-rpc=true #允许所有来源, web界面跨域权限需要 rpc-allow-origin-all=true #允许外部访问，false的话只监听本地端口 rpc-listen-all=true #RPC端口, 仅当默认端口被占用时修改 rpc-listen-port=6800 #最大同时下载数(任务数), 路由建议值: 3 max-concurrent-downloads=5 #断点续传 continue=true #同服务器连接数 max-connection-per-server=5 #最小文件分片大小, 下载线程数上限取决于能分出多少片, 对于小文件重要 min-split-size=10M #单文件最大线程数, 路由建议值: 5 split=10 #下载速度限制 max-overall-download-limit=0 #单文件速度限制 max-download-limit=0 #上传速度限制 max-overall-upload-limit=0 #单文件速度限制 max-upload-limit=0 #断开速度过慢的连接 #lowest-speed-limit=0 #验证用，需要1.16.1之后的release版本 #referer=* #文件保存路径, 默认为当前启动位置 dir=你的可道云储存地址 #文件缓存, 使用内置的文件缓存, 如果你不相信Linux内核文件缓存和磁盘内置缓存时使用 #disk-cache=0 #另一种Linux文件缓存方式 #enable-mmap=true #文件预分配, 能有效降低文件碎片, 提高磁盘性能. 缺点是预分配时间较长 file-allocation=prealloc file-allocation=prealloc # 禁用IPv6, 默认:false disable-ipv6=true 说明：

1、默认情况下只需要更改第一项 rpc-secret=token 这个即可，将 token 改成自己设置的秘钥（相当于通信秘钥），后期安装 web 前端界面需要用。

2、参考链接：Aria2 & YAAW 使用说明

##3、启动 Aria2

先测试： aria2c –conf-path=/etc/aria2/aria2.conf 如无错误后直接启动 aria2c –conf-path=/etc/aria2/aria2.conf -D ##4、安装 yaaw 前端控制界面

下载网页文件：https://github.com/binux/yaaw

直接放到网站目录下，即可访问

点击设置按钮，设置rpc 信息

http://token:自己设置的秘钥@IP地址:6800/jsonrpc

ip那里动态域名解析的填你的解析域名

保存即可使用
