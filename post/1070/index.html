Cloudreve对接onedrive搭建属于自己的网盘系统

发表于： 2020-06-24 分类于： 网络

介绍

Cloudreve能助您以最低的成本快速搭建公私兼备的网盘系统，简而言之就是想做一个像百度云一样的网盘系统！

改程序支持本机储存、七牛云、阿里云腾讯云oss等，还支持onedrive。前几种成本较高，但是onedrive成本低了，随手就能申请一个5T账号，别说还能升级25T了！这里教大家怎么搭建并且对接onedrive网盘。

支持的功能

Cloudreve 新版V3特性：

  •  支持本机、从机、七牛、阿里云 OSS、腾讯云 COS、又拍云、OneDrive (包括世纪互联版) 作为存储端
  • 上传/下载 支持客户端直传，支持下载限速
  •  可对接 Aria2 离线下载
  •  在线 压缩/解压缩、多文件打包下载
  •  覆盖全部存储策略的 WebDAV 协议支持
  •  拖拽上传、目录上传、流式上传处理
  •  文件拖拽管理
  •  多用户、用户组
  • 创建文件、目录的分享链接，可设定自动过期
  • 视频、图像、音频、文本、Office 文档在线预览
  • 自定义配色、黑暗模式、PWA 应用、全站单页应用
  •  All-In-One 打包，开箱即用

## 开工
  • 首先下载源码：https://github.com/cloudreve/Cloudreve/releases
  • ssh自己的服务器，输入
arch 输出结果x86_64代表amd64；aarch64代表arm64

下载链接处找到对应的版本，下载。一般服务器选择第一个就可以了。

在宝塔面板上添加网站，然后上传刚下载的源码，解压。然后防火墙开放5212端口

在ssh上cd到自己的网站目录，每个人的网站目录不同，不要一律照抄 cd /www/wwwroot/cloudreve.org(改为你自己的网站目录）

输入以下命令

chmod +x ./cloudreve
./cloudreve

访问你的ip/域名:5212就可以使用了

 

这里有三个问题:

  1. 数据库是用的自带的SQLite，要改为mysql
  2. 带端口要去掉
  3. 关闭SSH后，程序会停止运行
### SQLite改为mysql 新建一个mysql数据库，将相关信息添加在目录下的conf.ini文件里，命令行（./cloudreve）重启进程，获得新账号密码 [Database] ; 数据库类型，目前支持 sqlite | mysql Type = mysql ; 用户名 User = cloudreve ; 密码 Password = cloudreve ; 数据库地址 Host = 127.0.0.1 ; 数据库名称 Name = cloudreve ; 数据表前缀 TablePrefix = cd ## 去端口 这里只要反向代理就可以了

关闭SSH程序会停止运行

添加进程守护，在宝塔安装Supervisor管理器

 

获取网站目录和程序路径

添加守护进程

关闭SSH或者重启服务器，之后再Supervisor管理器中其中进程就可以了。

对接onedrive网盘

我们在管理面板-储存策略中添加储存策略，会提示添加onedrive是需要支持https的。在反向代理后，会提示不能开始SSL，这里我们只需要先开启SSL，再反向代理就可以了。
